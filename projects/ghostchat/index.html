Welcome to Ghost Chat!

Ghost Chat is a web application specifically designed for quick and secure interactions. Have you ever wanted to send a quick message but worry about the security of that message? That's where we come into play! All you have to do is create a room. Creating a room will generate a link you can give to another person. You post your message in the room, set an expiration time, and that's it.

Features

  • Secure - all messages are encrypted
  • No Saved Data - No sign up required and no saved messages. All messages expire after a set amount of time
  • Easy - In a matter of a few buttons and seconds you'll have a room setup with messages going to a designated person
